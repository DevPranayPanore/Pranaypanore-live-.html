About Memory and It's Type

  1. RAM (Random Access Memory):

    2. Function

    RAM acts as the computer's short-term memory, providing a space for the CPU to quickly access data and instructions needed for currently running programs and the operating system.

    Volatile:

    RAM is volatile, meaning its contents are erased when the computer is turned off or loses power.

    Speed:

    RAM is designed for speed, allowing the CPU to access data rapidly. 

    Types:

    Common types of RAM include DRAM (Dynamic RAM) and SRAM (Static RAM). 

    Examples:

    RAM is used for things like loading applications, browsing the internet, and editing documents.


  2. ROM (Read-Only Memory):

    3. Function:

    ROM stores essential instructions and data that are needed for the computer to boot up and operate, such as the BIOS (Basic Input/Output System).

    Non-volatile:

    ROM is non-volatile, meaning its contents are not lost when the power is turned off.

    Speed:

    ROM is slower than RAM. 

    Types:

    ROM can be mask ROM (manufactured with the data already stored), PROM (programmable only once), EPROM (erasable and programmable), and EEPROM (electrically erasable and programmable).

    Examples:

    ROM is used in computers, game consoles, and other devices to store firmware and startup instructions.
